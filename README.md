## 💎 Core Highlights

- **Executive "Overview" Tab**: Specifically designed for HR efficiency, providing a high-impact summary of technical stack and expertise at a glance.
- **Surgical Brand Identity**: Custom `[ munna.dev ]` wordmark with unified indigo gradients and micro-interactions.

## 🛠️ Tech Stack

- **Framework**: [Next.js 16 (App Router)](https://nextjs.org/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) (Modernized syntax)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Contact Engine**: [Web3Forms](https://web3forms.com/) (No-backend required)
- **Fonts**: Monospace for branding, Inter for readability.

## 🚀 Getting Started

### Prerequisites
- Node.js 20+
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/munnamiiraz/portfolio.git
   ```
2. Navigate to the client directory:
   ```bash
   cd client
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Set up environment variables:
   Create a `.env.local` file and add your Web3Forms access key:
   ```env
   NEXT_PUBLIC_WEB3FORMS_ACCESS_KEY=your_access_key_here
   ```
5. Run the development server:
   ```bash
   npm run dev
   ```

## 🌍 Deployment

This portfolio is optimized for **Vercel**. 
1. Push your code to GitHub.
2. Link the repository to Vercel.
3. Add the `NEXT_PUBLIC_WEB3FORMS_ACCESS_KEY` in the environment variables settings.
4. Deploy!
